How Publishing Works

Option A — Content team (no Terminal needed)
  1. Write post above → click "Add to Blog JSON" → file downloads
  2. Go to github.com → bite-dental-site → blog-posts.json ↗
  3. Click the pencil ✏️ icon (top right of file view)
  4. Select all text → paste in the downloaded JSON
  5. Click "Commit changes" → blog updates in ~2 min ✓
Option B — Dr. Shrey via Terminal (Mac)
  1. Download JSON → replace netlify_deploy/content/blog-posts.json
  2. Run:
    cd /Users/shreydhawan/Desktop/Bite_rebrand
    git add netlify_deploy/content/blog-posts.json
    git commit -m "Add blog post" && git push
💡 The blog reads directly from GitHub, so it works on any device — no server needed. Either method above updates the live blog within ~2 minutes.
